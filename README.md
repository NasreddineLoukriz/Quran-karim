#
 بعون الله وحفظه  تم برمجة هذا التطبيق باستخدام اطار فلاتر flutter البرنامج يعمل على كافة المنصات
 
### الخصائص 

* [x] Android iOS  linux windows macos ----- اندرويد اي او اس لينكس ويندوز ماك اواس

  * [x] high-definition pictures ----- صور عالية الدقة 
  * [x] Search in the index of the surahs Al-Ahzab ----- البحث في فهرس السور الأحزاب
  * [x] Search the entire Qur'an ----- البحث في القرآن كاملا 
  * [x] save bookmark -----حفظ علامة التوقف 
  * [x] Horizontal and vertical mode ----- الوضع الأفقي والعمودي
  * [ ] Interpretation of the Koran -----تفسير القرآن
